ColaJobs

Apresento o projeto ColaJobs, uma plataforma inovadora desenvolvida como parte da disciplina de Back End no Senac Penha. O nosso objetivo principal foi criar uma solução eficiente para conectar programadores talentosos a empresas em busca de contratação. Para isso, utilizamos uma variedade de tecnologias e ferramentas.

No desenvolvimento do ColaJobs, empregamos as linguagens HTML5, CSS3 e JavaScript para construir uma interface intuitiva e atraente. Além disso, utilizamos o framework Bootstrap 5 para garantir a responsividade e a adaptabilidade do site em diferentes dispositivos. Para a parte do servidor, utilizamos PHP, uma linguagem de programação amplamente usada e versátil. E para armazenar e gerenciar os dados, contamos com o poderoso banco de dados relacional MariaDB.

O ColaJobs representa uma solução simples e gratuita para facilitar o contato direto entre programadores e empresas contratantes. Nosso objetivo é proporcionar oportunidades de trabalho e conexões valiosas no mercado de TI. Com o ColaJobs, programadores talentosos têm a chance de expor suas habilidades e experiências diretamente para empresas em busca de profissionais qualificados. Ao mesmo tempo, as empresas podem encontrar os candidatos mais adequados para suas necessidades de contratação.

Estamos entusiasmados com o potencial do ColaJobs e confiantes de que essa plataforma contribuirá para a formação de parcerias profissionais bem-sucedidas e oportunidades de crescimento para todos os envolvidos.

PetsAmore

Durante o primeiro semestre, desenvolvemos um projeto de uma página web voltada para a doação de animais de estimação.

Para a criação dessa página, utilizamos tecnologias fundamentais no desenvolvimento web, como HTML, CSS e JavaScript. O HTML foi empregado para a estruturação dos elementos da página, enquanto o CSS foi responsável pela estilização, tornando-a atrativa e intuitiva para os usuários. Além disso, utilizamos o pré-processador SASS para facilitar a escrita e organização do código CSS.

O JavaScript foi implementado para adicionar interatividade à página. Essas tecnologias em conjunto proporcionaram uma experiência fluída e amigável para os usuários que desejam adotar ou doar animais de estimação.

Nosso principal objetivo com esse projeto foi criar uma plataforma eficiente e confiável para conectar pessoas interessadas em adotar animais de estimação com aquelas que estão procurando um novo lar para seus amiguinhos peludos. Acreditamos que a tecnologia pode desempenhar um papel importante nessa causa, facilitando o processo de adoção e promovendo o bem-estar animal.

Fala no Mic

O projeto Fala no Mic foi resultado de uma colaboração entre mim e meus colegas de classe, com o objetivo principal de desenvolver uma plataforma inovadora e dedicada à comunidade de hip-hop. Nosso foco foi proporcionar uma experiência de usuário (UX) fluida e intuitiva, valorizando a interação e a facilidade de uso.

Durante o processo de aprendizado, tive a oportunidade de trabalhar com o Firebase em um projeto recente, que se tornou uma aplicação essencial para os amantes do hip-hop em nossa cidade. Por meio dela, buscamos facilitar o acesso a eventos, shows e batalhas de rap locais, oferecendo informações atualizadas e um sistema de cadastro eficiente para os organizadores.

Para construir essa plataforma, utilizamos tecnologias avançadas como o React Native, garantindo uma interface moderna e responsiva. Além disso, aproveitamos os recursos do Firebase como banco de dados em nuvem para o armazenamento seguro e confiável dos dados. Também aplicamos técnicas de UX/UI para aprimorar a navegabilidade e proporcionar uma experiência agradável e satisfatória aos usuários.

Estamos orgulhosos do resultado alcançado e confiantes de que o Fala no Mic será uma ferramenta indispensável para a comunidade hip-hop, conectando artistas, fãs e eventos de forma única e impactante.

Agência Colors

O projeto em questão foi desenvolvido com base nos conhecimentos adquiridos durante o curso de Informática para Internet do Senac Penha. Utilizamos as linguagens HTML e CSS para criar uma experiência visual atraente e envolvente para os usuários.

Uma das principais características do projeto é o uso do efeito parallax, que proporciona uma sensação de profundidade e movimento à medida que o usuário rola a página. Esse efeito foi implementado com o objetivo de oferecer uma experiência diferenciada e agradável aos visitantes.

Durante o desenvolvimento, demos grande importância à legibilidade e à facilidade de leitura. Utilizamos boas práticas de design e layout para garantir que o conteúdo fosse apresentado de maneira clara e compreensível. Além disso, buscamos criar uma interface intuitiva, facilitando a navegação e a interação dos usuários com o projeto.

O projeto foi uma excelente oportunidade para colocar em prática os conhecimentos e habilidades adquiridos ao longo do curso. Sob a orientação do Professor Klaibert Miranda, pude explorar propriedades avançadas do HTML e CSS, ampliando meu domínio dessas linguagens e aprimorando minha capacidade de criação de projetos web.

Confeitaria ChocoNaty

O Projeto Choconaty utiliza uma variedade de tecnologias para criar um website atrativo, interativo e com ótima experiência do usuário. O HTML, CSS e JavaScript fornecem os fundamentos para estruturação, estilo e interatividade, enquanto as animações JavaScript adicionam um toque especial de dinamismo. Essas tecnologias são escolhidas levando em consideração a acessibilidade do site.

A aplicação de princípios de UX, aliada às tecnologias mencionadas, garante uma navegação intuitiva para todos os usuários, independentemente de suas habilidades e necessidades específicas. Ao desenvolver o projeto Choconaty, são implementadas práticas de design centradas no usuário, como layouts claros e intuitivos, menus de navegação bem estruturados e feedbacks visuais adequados. Isso proporciona uma experiência agradável e eficiente ao explorar o website.

Além disso, a compressão de imagens é uma técnica importante para otimizar o desempenho do site. Ao reduzir o tamanho dos arquivos de imagem por meio de técnicas de compressão adequadas, o tempo de carregamento das páginas é reduzido, melhorando a velocidade geral do site. Isso beneficia todos os usuários, especialmente aqueles que possuem conexões de internet mais lentas ou estão acessando o site por dispositivos móveis.

Ao unir essas tecnologias e práticas, o Projeto Choconaty busca oferecer não apenas um website visualmente atraente e interativo, mas também uma experiência de navegação fluida, intuitiva e acessível. A combinação de HTML, CSS, JavaScript, animações JavaScript, princípios de UX e compressão de imagens resulta em um site que promove e vende produtos de chocolate de alta qualidade de forma eficaz e agradável para todos os usuários.
