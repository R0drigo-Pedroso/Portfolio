    ColaJobs

    Apresento o projeto ColaJobs, uma plataforma que conecta programadores e empresas em busca de contratação. Desenvolvido como parte da UC de Back End no Senac Penha, o projeto foi realizado com HTML5, CSS3, JavaScript, Bootstrap 5, PHP e MariaDB. Agradeço à equipe incrível que participou desse projeto e à orientação dos professores envolvidos. O ColaJobs é uma solução simples e gratuita para facilitar o contato direto entre profissionais e empregadores, proporcionando oportunidades e conexões no mercado de trabalho.

    PetsAmore

    Durante o primeiro semestre, desenvolvemos um projeto de uma página web voltada para a doação de animais de estimação.

    Para a criação dessa página, utilizamos tecnologias fundamentais no desenvolvimento web, como HTML, CSS e JavaScript. O HTML foi empregado para a estruturação dos elementos da página, enquanto o CSS foi responsável pela estilização, tornando-a atrativa e intuitiva para os usuários. Além disso, utilizamos o pré-processador SASS para facilitar a escrita e organização do código CSS.

    O JavaScript foi implementado para adicionar interatividade à página. Essas tecnologias em conjunto proporcionaram uma experiência fluída e amigável para os usuários que desejam adotar ou doar animais de estimação.

    Nosso principal objetivo com esse projeto foi criar uma plataforma eficiente e confiável para conectar pessoas interessadas em adotar animais de estimação com aquelas que estão procurando um novo lar para seus amiguinhos peludos. Acreditamos que a tecnologia pode desempenhar um papel importante nessa causa, facilitando o processo de adoção e promovendo o bem-estar animal.
